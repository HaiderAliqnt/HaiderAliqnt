# About Me:
I'm a fourth-semester Computer Science student with a strong focus on cybersecurity<br>and penetration testing. My approach to learning security is hands on: I build systems to <br>understand how they work, then learn how to break and secure them. Rather than just <br>consuming theory, I believe in getting my hands dirty setting up vulnerable environments, <br>exploiting them, and understanding the attacker's mindset from the inside out. I also work <br>with web development, not just to build applications, but to deeply understand how web <br>technologies work so I can identify and exploit their vulnerabilities. <br>The best way to secure a system is to know exactly how an attacker would break it.
